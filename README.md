# KDKYS - Kurumsal Doküman ve Karar Yönetim Sistemi

## Proje Hakkında

KDKYS (Kurumsal Doküman ve Karar Yönetim Sistemi), kurum içi doküman paylaşımı, görev yönetimi, toplantı takibi ve personel iletişimini merkezi bir yapı altında toplamak amacıyla geliştirilmiş web tabanlı bir yönetim sistemidir.

Bu proje Yönetim Bilişim Sistemleri bölümü kapsamında ASP.NET Core MVC mimarisi kullanılarak geliştirilmiştir.

---

## Kullanılan Teknolojiler

- ASP.NET Core MVC
- Entity Framework Core
- SQLite
- HTML5
- CSS3
- JavaScript
- Bootstrap
- JWT Authentication
- Docker
- GitHub

---

## Sistem Özellikleri

### Kullanıcı Yönetimi

- Kullanıcı kayıt işlemleri
- Kullanıcı giriş işlemleri
- Rol bazlı yetkilendirme
- Yönetici ve Personel rolleri

### Doküman Yönetimi

- Doküman yükleme
- Doküman indirme
- Doküman paylaşımı
- Dahili / Harici kategorileri
- Doküman güncelleme
- Doküman silme

### Görev Yönetimi

- Görev oluşturma
- Personel atama
- Görev durum takibi
- Öncelik seviyeleri
- Verimlilik hesaplama

### Toplantı Yönetimi

- Toplantı oluşturma
- Toplantı notları
- Toplantı kararları
- Notları göreve dönüştürme

### Mesajlaşma Sistemi

- Kullanıcılar arası mesajlaşma
- Gelen kutusu
- Gönderilen mesajlar

### Bildirim Sistemi

- Yeni görev bildirimi
- Yeni toplantı bildirimi
- Doküman paylaşım bildirimi
- Mesaj bildirimi

### Dashboard

- Toplam kullanıcı sayısı
- Toplam doküman sayısı
- Bekleyen görevler
- Tamamlanan görevler
- Personel verimlilik analizi
- Excel rapor oluşturma

---

## Yetkilendirme Yapısı

### Yönetici

- Kullanıcı ekleyebilir
- Kullanıcı silebilir
- Kullanıcı güncelleyebilir
- Görev oluşturabilir
- Toplantı oluşturabilir
- Tüm sistem verilerini görüntüleyebilir

### Personel

- Kendi görevlerini görüntüleyebilir
- Kendi notlarını düzenleyebilir
- Kendi dokümanlarını yönetebilir
- Toplantı notu ekleyebilir
